Папки проекта

Папка "data_bases" предназначена для хранения баз данных.
Однако сами бд не загружаются на github в целях конфиденциальности.

Папка "documentation" предназначена для хранения документации проекта.
В файле "main.txt" хранятся общая информация о проекте.
В остальных файлах хранится информация об одноименных файлах проекта.
Например: файл "db_schedule_changer.txt" хранит информацию о файле "db_schedule_changer.py"

Папка "handlers" предназначена для обработчиков.
В ней хранятся скрипты, обрабатывающие обновления от Telegram API.

Папка "parsing" предназначена для ...

Папка "scripts" предназначена для неклассифицированных скриптов.
Например, файлы начинающиеся с "db_" отвечают за работу с базами данных.

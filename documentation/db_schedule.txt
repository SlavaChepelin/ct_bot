Документация файла "db_schedule.py"


Функция "add_schedule_change" является асинхронной.

Функция принимает 6 значений:
Параметр "group": принимает число — номер группы в формате "ABB", где A — курс, BB номер, например "139" это M3139.
Параметр "date": принимает строку — дату в формате "yyyy.mm.dd", например "2024.09.03".
Параметр "number": принимает число — порядковый номер пары начиная с 1, например пара с номером 2 идет с 10:00 до 11:30.
Параметр "subject": принимает строку — название предмета как в таблице расписания, например "АиСД".
Параметр "lesson_type": принимает строку — тип пары как в таблице расписания, например "лек".
Параметр "auditorium": принимает строку — номер аудитории, например "2137".

Параметры записываются в базу данных "db_schedule.db" в лист "Changes".

Функция возвращает "None".


Функция "add_schedule_change" является асинхронной.

Функция принимает 1 значение:
Параметр "date": принимает строку — дату в формате "yyyy.mm.dd", например "2024.09.03".

Функция извлекает данные из базы данных "db_schedule.db" листа "Changes" при совпадении параметра "data".

Функция возвращает массив кортежей с параметрами перечисленными выше.
Формат вывода:
[(group, date, number, subject, lesson_type, auditorium), ...]
Если данные не найдены, будет выведен None.
